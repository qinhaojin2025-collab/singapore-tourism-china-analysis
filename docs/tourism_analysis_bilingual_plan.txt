题目 / Title
中国客源市场变化与新加坡旅游业复苏分析：基于月度入境旅客、酒店入住率与逗留时间的多方法研究
Changes in the Chinese Source Market and Singapore's Tourism Recovery: A Multi-Method Study Based on Monthly Visitor Arrivals, Hotel Occupancy, and Length of Stay

整体思路 / Overall Logic
这份数据最适合做“月度旅游市场状态分析”。你的核心问题不是“游客个人画像”，而是“在疫情冲击与复苏过程中，中国游客变化是否和新加坡旅游市场表现有关”。
This dataset is best suited for a "monthly tourism market condition" study. Your core question is not about individual tourist profiles, but whether changes in Chinese visitor arrivals are associated with Singapore's tourism performance during the shock and recovery periods.

主分析变量 / Main Analytical Variables
- date：月份 / month
- visitor_arrivals：总入境旅客 / total visitor arrivals
- visitor_arrivals_china：中国入境旅客 / visitor arrivals from China
- hotel_occ：酒店客房入住率 / hotel room occupancy rate
- avg_stay_monthly：平均逗留时间（月） / average length of stay (monthly)

补充变量 / Supplementary Variables
- spend_per_capita：游客人均支出 / tourist expenditure per capita
- tourism_receipts：旅游业收入 / tourism receipts
- avg_stay_annual：平均逗留时间（年） / average length of stay (annual)

这些年度变量只适合做补充图表，不建议放进聚类和决策树。
These annual variables should only be used for supplementary charts, not for clustering or decision tree modeling.

数据预处理 / Data Preparation
- 删除前 29 行元数据，只保留真实观测值。
  Remove the first 29 metadata rows and keep only the actual observations.
- 删除 2015-12-01 这一行，因为月度主变量基本缺失。
  Remove 2015-12-01, because the main monthly variables are mostly missing there.
- 新增 china_share = visitor_arrivals_china / visitor_arrivals。
  Create china_share = visitor_arrivals_china / visitor_arrivals.
- 新增 year、month、quarter。
  Create year, month, and quarter.
- 新增 period：pre_covid、covid_shock、recovery。
  Create period: pre_covid, covid_shock, and recovery.

建议时间分段 / Suggested Period Definition
- pre_covid：2016-12 到 2020-01 / from 2016-12 to 2020-01
- covid_shock：2020-02 到 2021-12 / from 2020-02 to 2021-12
- recovery：2022-01 到 2026-01 / from 2022-01 to 2026-01

Part 2: 确认性数据分析 / Confirmatory Data Analysis
这一部分要回答“这些差异和关系是否显著”。
This part answers: "Are these differences and relationships statistically significant?"

建议设置 4 个假设 / Suggested Hypotheses
- H1：不同阶段的酒店入住率有显著差异。
  H1: Hotel occupancy differs significantly across periods.
- H2：不同阶段的中国游客占比有显著差异。
  H2: China's visitor share differs significantly across periods.
- H3：总入境旅客与酒店入住率呈显著正相关。
  H3: Total visitor arrivals are significantly positively correlated with hotel occupancy.
- H4：月均逗留时间与总入境旅客呈显著负相关。
  H4: Monthly average length of stay is significantly negatively correlated with total visitor arrivals.

建议方法 / Suggested Methods
- 如果数据近似正态，可用 ANOVA。
  If the data are approximately normal, use ANOVA.
- 如果不满足正态，可用 Kruskal-Wallis。
  If normality is not satisfied, use Kruskal-Wallis.
- 连续变量关系可用 Pearson 或 Spearman 相关分析。
  For continuous relationships, use Pearson or Spearman correlation.
- 可以补一个简单回归，以 hotel_occ 为因变量。
  You may add a simple regression with hotel_occ as the dependent variable.

这一部分的结论写法可以是：
A good writing style for this part is:
"结果表明，不同阶段的酒店入住率存在显著差异，且总入境旅客与酒店入住率呈显著正相关，说明旅游市场恢复与住宿市场修复具有同步性。"
"The results show that hotel occupancy differs significantly across periods, and total visitor arrivals are significantly positively associated with hotel occupancy, suggesting that tourism recovery and accommodation market recovery moved in tandem."

Part 3: 聚类分析 / Clustering Analysis
这一部分要回答“这些月份能否被分成几种典型市场状态”。
This part answers: "Can the months be grouped into several typical market states?"

聚类对象一定是“月份”。
The clustering unit must be the "month."

推荐聚类变量 / Recommended Clustering Variables
- visitor_arrivals
- china_share
- hotel_occ
- avg_stay_monthly

为什么用 china_share 而不是只用 china_visitors：
Why use china_share instead of only china_visitors:
- china_share 更能体现市场结构变化。
  china_share better captures market structure changes.
- 它可以说明中国市场在整体旅游中的权重。
  It shows the importance of the Chinese market within total tourism.

处理步骤 / Processing Steps
- 删除缺失月。
  Remove months with missing values.
- 对连续变量做标准化。
  Standardize continuous variables.
- 对 avg_stay_monthly 做温和异常值处理，因为疫情期偏高。
  Apply mild outlier treatment to avg_stay_monthly, because it spikes during the pandemic.
- 用肘部法和轮廓系数确定聚类数。
  Use the elbow method and silhouette score to determine the number of clusters.
- 建议先试 K=3。
  Start with K=3.

You can finally interpret the clusters as:

- Cluster 2：恢复过渡型月份 / recovery transition months
- Cluster 3：高景气型月份 / high-performance months

The key message in this part should be:
