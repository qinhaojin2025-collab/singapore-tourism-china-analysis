# Singapore Tourism China Market Analysis

This repository contains the cleaned data, supporting documents, and reproducible cleaning script for a course-style analysis of Singapore tourism recovery with a focus on the Chinese visitor market.

## Repository Structure

- `data/raw/`
  - Original source workbook used for the analysis.
- `data/processed/`
  - Cleaned and analysis-ready datasets for EDA, CDA, clustering, and decision tree analysis.
- `docs/`
  - Bilingual analysis plan and detailed data-cleaning documentation.
- `scripts/`
  - Python script used to clean and prepare the processed datasets.

## Main Files

- `data/processed/tourism_four_part_analysis_ready.xlsx`
  - Final all-in-one dataset prepared for the four analysis modules.
- `data/processed/tourism_four_part_analysis_ready.csv`
  - CSV version of the final analysis-ready dataset.
- `data/processed/tourism_monthly_clean.csv`
  - Clean monthly dataset for EDA, CDA, and clustering.
- `data/processed/tourism_decision_tree_ready.csv`
  - Decision-tree-ready dataset with class labels and train/test split.
- `docs/tourism_data_cleaning_detailed_report.md`
  - Detailed explanation of what was cleaned and why.
- `scripts/clean_tourism_data.py`
  - Script used to generate the cleaned outputs.
